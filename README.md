## Star

Star is a lightweight multithreaded concurrent framework for Lua.

## Build

For Linux:

```
git clone https://github.com/HYbutterfly/star.git
cd star
make
```

## Test

Run these:

```
./star conf.lua
```

## About Lua version

Star now uses lua 5.3.4
